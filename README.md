# Bank Customer Churn Analysis
 
**Tools**: Excel, MySQL

## Overview
This project analyzes bank customer churn using **Excel dashboards** and **SQL queries**.  
It answers key business questions such as churn by geography, gender, tenure, product count, and customer profiles.

## Repository Layout
- **data/**: raw and processed datasets
- **excel/**: Excel dashboard file
- **sql/queries/**: individual SQL scripts for each business question
- **reports/**: final report and figures
- **README.md**: project overview
